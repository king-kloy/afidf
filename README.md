# African International Documentary Festival

## Getting Started

```bash
 git clone https://github.com/king-kloy/afidf.git

 # change to the project directory
 cd afidf

 # install node modules or dev dependencies
 npm install

 # to start the client
 cd client
 npm install
 npm start

 # to start backend
 cd backend
 yarn
 yarn start

 # to both concurrently at the root
 npm start

```
